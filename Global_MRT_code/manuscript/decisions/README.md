# Analysis decision records

Short, dated notes documenting non-obvious analysis choices in this study: the
question, the decision, the reasoning, the supporting numbers, and how to
reproduce them. One file per decision. The point is that any co-author or reviewer
can later ask "why did you do X?" and find a self-contained, versioned answer
instead of relying on anyone's memory.

Format per file: `YYYY-MM-DD_short-slug.md`. Keep each to one screen where possible;
link to the script/output that produces the numbers so the record stays checkable.
